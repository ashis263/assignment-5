Frequently Asked Questions About Document Object Model

What is the Document Object Model (DOM)?

Document Object Model (DOM) is a interface or we can say, platform, which allows us to get the web's markup language's elements as object (HTML like object) by using javascript. We can easily create, remove or modify elements, elements's attributes, classes and style them by using Document Object Model.

How do you select an element from the DOM?

There are several ways to select an element using DOM. The most used way of selecting is by selecting the ID (document.getElementById('id')). We can use use document.querySelector('query-selector') which selects the first element which that matches the query. Both of them selects a single element. Also we can select multiple by using document.getElementsByClassName('class-name') which selects all the elements that matches the class name class-name, document.getElementsByTagName('tag-name') which selects all the elements that matches the tag tag-name and document.querySelectorAll('query-selector') which selects all the elements that matches the query selector query-selector

What is event delegation in the context of the DOM, and why is it useful?

The way of handling an event of an element by selecting it's parent element, then selecting parent's children and then handle the event is event delegation. If we want to use same event listener to multiple elements which are sibling to each other, we can simply delegate the event listener to their parent and then optimize our code by looping through the parent and write a single event listener. Thats why event delegation is useful.

How do you manipulate an element's attributes and styles using the DOM?

To manipulate an element's attribute we need to select the element first then use the setAttribute('attribute-name') or removeAttribute('attribute-name') method to add or remove attribute with the name attribute-name. document.getElementById('id').setAttribute('attribute-name') By using style.cssPropertyName = 'css-value' after selecting the element we can easily add or modify it's style. document.getElementById('id').style.cssPropertyName = 'css-value'
